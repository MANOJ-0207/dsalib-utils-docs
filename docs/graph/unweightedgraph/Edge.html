Edge

Type: Record

Description

Edge<K> is an immutable data record representing an edge between two nodes in a graph. It can represent both directed and undirected edges based on the isDirected flag.

For undirected edges, (u, v) is considered equal to (v, u). For directed edges, (u → v) is only equal to another edge with the same direction and nodes.

Fields

  • K u — the source node (or one endpoint in an undirected graph)
  • K v — the destination node (or the other endpoint in an undirected graph)
  • boolean isDirected — whether the edge is directed (true) or undirected (false)

Method Summary

Return Type Method Description Time Complexity
N/A Edge(K u, K v, boolean isDirected) Constructs a directed/ undirected edge from u to v based on isDirected flag O(1)
boolean equals(Object o) Compares this edge to another object. For directed edges, both direction and endpoints must match. For undirected edges, the order of endpoints does not matter. O(1)
int hashCode() Computes the hash code of the edge. For directed edges, the hash is order-sensitive. For undirected edges, it is order-insensitive to maintain symmetry. O(1)
String toString() Returns a string representation of the edge. Uses (u → v) for directed and (u — v) for undirected edges. O(1)

Basic Usage


// Undirected edge
Edge<Integer> e1 = new Edge<>(1, 2, false);
Edge<Integer> e2 = new Edge<>(2, 1, false);

System.out.println(e1.equals(e2)); // true
System.out.println(e1.hashCode() == e2.hashCode()); // true
System.out.println(e1); // (1 — 2)

// Directed edge
Edge<String> e3 = new Edge<>("A", "B", true);
Edge<String> e4 = new Edge<>("B", "A", true);

System.out.println(e3.equals(e4)); // false
System.out.println(e3); // (A → B)
