SimpleGraph

Type: Interface

Description

SimpleGraph<K> defines the contract for unweighted graph implementations, including operations like edge manipulation, traversal, reachability, and bridge detection.

Method Contracts

Return Type Method Description
void addEdge(K from, K to) Adds an edge between two nodes (auto-creates nodes if absent)
void removeEdge(K from, K to) Removes an edge if it exists
void bfs(K source, Consumer<K> action) Performs breadth-first traversal from source; given action is performed on each visited node
void dfs(K source, Consumer<K> action) Performs depth-first traversal from source; given action is performed on each visited node
List<K> getTopologicalSort() Returns topological order of nodes (only valid for DAGs)
Map<K, Integer> shortestDistances(K source) Returns shortest unweighted distances from source node
boolean isReachable(K from, K to) Checks if a path exists between two nodes
boolean isConnected() Checks if the graph is fully connected
boolean isBridgeEdge(K u, K v) Checks if the edge is a bridge (cut-edge)
boolean isBridgeNode(K node) Checks if the node is a bridge node (articulation point)
Set<Edge<K>> getBridgeEdges() Returns all bridge edges in the graph
Set<K> getBridgeNodes() Returns all articulation points (bridge nodes)

Available Implementations

  • AdjListSimpleGraph - Adjacency list-based implementation
  • AdjMatrixSimpleGraph - Adjacency matrix-based implementation
