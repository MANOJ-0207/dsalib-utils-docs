WeightedGraph

Type: Interface

Description

WeightedGraph<K> is an interface for representing weighted graphs. It supports both adjacency list and matrix-based implementations, and provides APIs for traversal, shortest path, MST, and SCC algorithms.

  • Supports: BFS, DFS, Dijkstra, Bellman-Ford, Floyd-Warshall
  • MST Algorithms: Prim’s, Kruskal’s
  • SCC: Kosaraju or Tarjan internally

Method Contracts

Return Type Method Description
void addEdge(from, to, weight) Adds a weighted edge between nodes
void removeEdge(from, to) Removes the edge between nodes
void bfs(start, action) Performs BFS from start
void dfs(start, action) Performs DFS from start
List<List<K>> getStronglyConnectedComponents() Returns all SCCs as lists
boolean isStronglyConnected() Returns true if all nodes are reachable from each other
int getSCCCount() Returns number of SCCs
Map<K, Integer> getSCCMap() Maps each node to its SCC ID
boolean isReachable(from, to) Checks if a path exists
int dijkstra(source, destination) Returns shortest path using Dijkstra's algorithm
Map<K, Integer> bellmanFord(source) Returns shortest paths using Bellman-Ford algorithm
Map<K, Integer> floydWarshall() Computes all-pairs shortest paths
int primsMST(start) Returns total weight of MST using Prim’s algorithm
int kruskalMST() Returns total weight of MST using Kruskal’s algorithm
void printGraph() Prints current graph structure

Implementations

  • AdjListWeightedGraph – Adjacency list-based implementation
  • AdjMatrixWeightedGraph – Matrix-based implementation

Basic Usage


// Using the weighted graph interface
WeightedGraph<String> graph = new AdjListWeightedGraph<>();

graph.addEdge("A", "B", 4);
graph.addEdge("B", "C", 2);

// Dijkstra shortest path
int shortest = graph.dijkstra("A", "C");
System.out.println(shortest); // Output: 6

// Get all SCCs
List<List<String>> sccs = graph.getStronglyConnectedComponents();
System.out.println(sccs);
