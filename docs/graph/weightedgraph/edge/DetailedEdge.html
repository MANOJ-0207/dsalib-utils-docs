DetailedEdge

Type: Record

Description

DetailedEdge<K> represents a weighted directed edge in a graph from a source node to a dest node, along with an integer weight.

This class is ideal for algorithms such as:

  • Dijkstra's Algorithm
  • Bellman-Ford
  • Kruskal's MST
  • Prim's MST (less common due to directionality)

Being a Java record, it provides:

  • Immutable structure
  • Thread safety
  • Auto-generated equals(), hashCode(), and toString()

Fields

  • K source — the starting node of the directed edge
  • K dest — the ending node of the directed edge
  • int weight — the weight associated with the edge

Method Summary

Return Type Method Description Time Complexity
N/A DetailedEdge(K source, K dest, int weight) Creates a new weighted directed edge from source to dest O(1)
String toString() Returns a string in the format: source --weight-→ dest O(1)
boolean equals(Object o) Checks equality based on all fields: source, dest, and weight O(1)
int hashCode() Generates hash based on all fields O(1)

Basic Usage


DetailedEdge<String> e1 = new DetailedEdge<>("A", "B", 5);
DetailedEdge<String> e2 = new DetailedEdge<>("B", "C", 10);

System.out.println(e1);  // A --5-→ B
System.out.println(e2);  // B --10-→ C
