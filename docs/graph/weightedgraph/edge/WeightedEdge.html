WeightedEdge

Type: Record

Description

WeightedEdge<K> represents a weighted edge pointing to a target node with a specified weight. It is most commonly used in adjacency lists of weighted graphs.

This structure is used in:

  • Dijkstra's Algorithm
  • Prim's Minimum Spanning Tree
  • Any graph traversal involving weights to neighbors

As a record, this class offers:

  • Immutable and thread-safe design
  • Auto-generated equals(), hashCode(), and toString()
  • Custom equality: only considers the target node
  • Comparable — ordered by weight (ascending)

Fields

  • K node — the target node this edge points to
  • int weight — the weight of the edge

Method Summary

Return Type Method Description Time Complexity
N/A WeightedEdge(K node, int weight) Creates a weighted edge pointing to a target node O(1)
int compareTo(WeightedEdge<K> other) Compares this edge with another based on weight O(1)
String toString() Returns a string in the format: node (wt: weight) O(1)
boolean equals(Object obj) Returns true if the other object points to the same node (ignores weight) O(1)
int hashCode() Generates hash code based on the target node O(1)

Basic Usage


WeightedEdge<String> e1 = new WeightedEdge<>("B", 3);
WeightedEdge<String> e2 = new WeightedEdge<>("B", 7);
WeightedEdge<String> e3 = new WeightedEdge<>("C", 2);

// Equality check (ignores weight)
System.out.println(e1.equals(e2)); // true

// Comparison by weight
System.out.println(e3.compareTo(e1)); // -1 (since 2 < 3)

// Output
System.out.println(e1); // B (wt: 3)
