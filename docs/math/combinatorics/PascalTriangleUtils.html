PascalTriangleUtils

Type: Utility Class

Description

PascalTriangleUtils is a lightweight utility class for working with Pascal's Triangle in combinatorics. It supports:

  • Building Pascal’s triangle up to n rows
  • Fetching any nth row directly
  • Computing individual binomial coefficients: C(n, k)
  • Finding the peak value in a row

All computations are performed in long internally to reduce overflow risk, but returned as int with appropriate safety checks.

Constant Summary

  • MAX_SAFE_N = 33 — Upper bound to avoid integer overflow (nCr(33, 16) <= 2³¹−1).

Method Summary

Return Type Method Description Time Complexity
List<List<Integer>> buildTriangle(int n) Builds full Pascal’s triangle up to n rows. O(n²)
List<Integer> getNthRow(int n) Generates the nth row of Pascal’s triangle directly. O(n)
int getValue(int row, int col) Computes C(row, col) directly without full triangle. O(k)
int getPeak(int row) Returns the maximum value in the given row. O(row)

Detailed Method Descriptions

buildTriangle(n)

Constructs Pascal’s triangle with n rows (0-based). Row i contains i + 1 elements.

Throws ArithmeticException if any value exceeds Integer.MAX_VALUE. 


List<List<Integer>> triangle = PascalTriangleUtils.buildTriangle(5);
System.out.println(triangle);
// [
//   [1],
//   [1, 1],
//   [1, 2, 1],
//   [1, 3, 3, 1],
//   [1, 4, 6, 4, 1]
// ]

getNthRow(n)

Computes the nth row (0-indexed) using iterative binomial logic without building the full triangle. 


List<Integer> row7 = PascalTriangleUtils.getNthRow(7);
// [1, 7, 21, 35, 35, 21, 7, 1]

getValue(row, col)

Computes the binomial coefficient C(row, col), or returns 0 if col is out of bounds. Computed using multiplicative formula: 


int val = PascalTriangleUtils.getValue(5, 2); // 10

getPeak(row)

The largest value in Pascal’s Triangle row is at row / 2. 


int peak = PascalTriangleUtils.getPeak(6); // 20

Mathematical Notes

  • Binomial Formula: C(n, k) = n! / (k!(n − k)!)
  • Each row starts and ends with 1
  • Overflow protection is enforced via MAX_SAFE_N
  • Symmetry: C(n, k) = C(n, n − k)

Quick Example


System.out.println(PascalTriangleUtils.getValue(10, 3));    // 120
System.out.println(PascalTriangleUtils.getPeak(10));        // 252
System.out.println(PascalTriangleUtils.getNthRow(5));       // [1, 5, 10, 10, 5, 1]
