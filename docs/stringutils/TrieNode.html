TrieNode

Type: Package-Private Class

Description

TrieNode is a helper node class used internally by the Trie data structure. It represents a single character in a word and maintains metadata useful for prefix-based operations.

Node Structure

  • children: Map of characters to their corresponding child nodes
  • isEndOfWord: Flag indicating if this node marks the end of a valid word
  • wordCount: Number of complete words ending at this node
  • prefixCount: Number of words passing through this node as a prefix

Constructor

Constructor Description Time Complexity
