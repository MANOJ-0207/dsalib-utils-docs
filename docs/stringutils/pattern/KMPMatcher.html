KMPMatcher

Type: Concrete Class

Description

KMPMatcher is a Knuth-Morris-Pratt (KMP) pattern matcher that extends AbstractPatternMatcher. It efficiently finds all occurrences of a pattern in text using preprocessed Longest Prefix Suffix (LPS) information, achieving linear time complexity by avoiding unnecessary character comparisons.

Use Cases

  • Text processing applications requiring efficient pattern search
  • DNA sequence analysis and bioinformatics
  • Large document search and indexing systems
  • Code editors for find operations on large files
  • Log file analysis and real-time text streaming
  • Data mining and text analytics

Method Summary

Return Type Method Description Time Complexity
List<Integer> search(String text, String pattern) Inherited from AbstractPatternMatcher with validation O(N + M)
List<Integer> search(char[] text, char[] pattern) Inherited from AbstractPatternMatcher with validation O(N + M)
List<Integer> doSearch(char[] text, char[] pattern) Core KMP algorithm implementation O(N + M)

Algorithm Details

The KMP algorithm works in two phases:

  • Preprocessing: Builds LPS (Longest Prefix Suffix) array in O(M) time
  • Searching: Uses LPS array to avoid redundant comparisons in O(N) time

Space Complexity: O(M) for the LPS array, where M is pattern length.

Sample Usage

Basic Pattern Matching


KMPMatcher kmpMatcher = new KMPMatcher();

// Find all occurrences of "hello" in text
String text = "hello world hello there hello again";
String pattern = "hello";
List<Integer> matches = kmpMatcher.search(text, pattern);
System.out.println("Matches found at positions: " + matches);
// Output: [0, 12, 24]

DNA Sequence Analysis


KMPMatcher dnaSearcher = new KMPMatcher();

// Search for specific DNA sequence
String dnaSequence = "ATCGATCGATCGATGCATGCATGCATCGATCG";
String targetGene = "ATGC";
List<Integer> genePositions = dnaSearcher.search(dnaSequence, targetGene);
System.out.println("Gene sequence found at: " + genePositions);
// Output: [15, 19, 23]

Large Text Processing


KMPMatcher textProcessor = new KMPMatcher();

// Process large documents efficiently
String largeDocument = "The quick brown fox jumps over the lazy dog. " +
                      "The fox is quick and the dog is lazy...";
String searchTerm = "the";

List<Integer> occurrences = textProcessor.search(largeDocument.toLowerCase(), searchTerm);
System.out.println("Found '" + searchTerm + "' " + occurrences.size() + " times");
System.out.println("At positions: " + occurrences);

Character Array Usage


KMPMatcher charMatcher = new KMPMatcher();

// Work directly with character arrays for performance
char[] textArray = "abcabcabcabc".toCharArray();
char[] patternArray = "abcabc".toCharArray();

List<Integer> results = charMatcher.search(textArray, patternArray);
System.out.println("Pattern matches at: " + results);
// Output: [0, 3, 6]

Repeating Pattern Detection


KMPMatcher patternDetector = new KMPMatcher();

// Detect overlapping patterns
String repeatingText = "aaaaaaa";
String overlappingPattern = "aaa";

List<Integer> overlappingMatches = patternDetector.search(repeatingText, overlappingPattern);
System.out.println("Overlapping matches: " + overlappingMatches);
// Output: [0, 1, 2, 3, 4] - KMP handles overlapping patterns correctly
