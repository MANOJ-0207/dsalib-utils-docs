PatternMatcher

Type: Interface

Description

PatternMatcher is an interface for string pattern matching algorithms. It provides overloaded methods to support both String and char[] inputs. Implementations should return all starting indices where the pattern occurs in the text.

Method Contracts

Return Type Method Description Expected Behavior
List<Integer> search(String text, String pattern) Searches for pattern in text string Returns list of starting indices; throws IllegalArgumentException if inputs are null
List<Integer> search(char[] text, char[] pattern) Searches for pattern in character array Returns list of starting indices; throws IllegalArgumentException if inputs are null

Available Implementations

  • AbstractPatternMatcher - Abstract base class with input validation

Basic Usage


// Using any implementation
PatternMatcher matcher = new KMPPatternMatcher();

// Search with strings
List<Integer> positions = matcher.search("hello world hello", "hello");
System.out.println(positions); // Output: [0, 12]

// Search with character arrays
char[] text = "abcdefabcdef".toCharArray();
char[] pattern = "def".toCharArray();
List<Integer> charResults = matcher.search(text, pattern);
System.out.println(charResults); // Output: [3, 9]
